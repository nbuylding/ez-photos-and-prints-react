
# EZ Photos & Prints

A responsive website for a photo printing store that includes a list of products that connects to a Whatsapp API and an FAQ for services the store provides.
## Built With

- HTML5
- CSS3
- JavaScript
- React
- [ElfSight](https://elfsight.com/)
